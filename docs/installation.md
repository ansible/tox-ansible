# Installation

Getting started with tox-ansible is as simple as:

```bash
pip install tox-ansible
```

## Dependencies

`tox-ansible` will install additional dependencies if necessary:

- `tox` version 4.0 or greater.
- `pytest-ansible` version 3.1.0 or greater.
- `pytest`
- `pytest-xdist`
- `pyyaml`
